🎬 API de Filmes
Uma API simples para gerenciamento de filmes, permitindo listar, adicionar, atualizar e remover filmes de uma base de dados.

📌 Onde podemos:
- Listar todos os filmes
- Buscar filme por ID ou nome
- Adicionar novo filme
- Atualizar informações de um filme
- Remover filmes

🚀 Tecnologias Utilizadas:
- Node.js
- Express
